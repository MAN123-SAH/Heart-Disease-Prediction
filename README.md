# ❤️ Heart Disease Prediction App

A Machine Learning based web application that predicts the risk of heart disease using patient health parameters. The application is built using **Python**, **Scikit-learn**, and **Streamlit** to provide instant and user-friendly predictions.

---

## 📌 Project Overview

Heart disease is one of the leading causes of death worldwide. Early detection can help in timely medical consultation and preventive care.

This project predicts the likelihood of heart disease based on medical attributes such as:

* Age
* Gender
* Blood Pressure
* Cholesterol Level
* Heart Rate
* Chest Pain Type
* Blood Sugar and other clinical parameters

The system provides an instant result indicating whether the person is at **High Risk** or **Low Risk**.

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/MAN123-SAH/Heart-Disease-Prediction.git
cd Heart-Disease-Prediction
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the Streamlit app

```
streamlit run app.py
```

The app will open in your browser at:

```
http://localhost:8501
```
